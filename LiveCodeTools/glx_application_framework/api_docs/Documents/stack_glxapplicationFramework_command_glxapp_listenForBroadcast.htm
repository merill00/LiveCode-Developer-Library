command glxapp_listenForBroadcast

Registers an object to listen for broadcasts that another object can send.

command glxapp_listenForBroadcast pObject, pBroadcastTypes, pBroadcastTarget, pMessage
Parameters
TypeNameDescription
pObject A reference to an object holding broadcast information or "global". This parameter refers to the broadcaster.
pBroadcastTypes A comma delimited list of types of broadcasts the listener wants to hear about.
pBroadcastTarget A reference to the object that will listen for broadcasts from pObject.
pMessage The message that will be sent to pBroadcastTarget. If left empty then the default message for the broadcast type will be sent.
Return

Error message

See also
