Tell Framework to Manage Application Window

Now that I have a working application let's look at how to tell the framework to manage the Article Maestro window. The framework will send special messages and manage the size and position of the window across sessions.

Edit Article Maestro Properties

I right-click on the Article Maestro stack in the Application Browser and choose Property Inspector from the menu.

Add Custom Property Set

I click on the + button to add a new custom property set.

I name the custom property set uGLXApp.

Add Custom Property

I click on the + button to add a custom property to the uGLXApp custom property set.

I name the property manage window.

I set the value to true.
