Installing the Plugin

This lesson will show you how to install the GLX Application Framework IDE plugin. Plugins are installed in the plugins folder of your Revolution User Extensions folder.

Locate Plugin Stack

You can locate the plugin stack in the ./plugin folder of the GLX Application Framework distribution folder. If you are using Revolution 3.5 or later then choose the plugin in the Revolution 3.5 or later folder. This version of the plugin uses a Data Grid and is more user friendly. This is the version of the plugin that will be updated moving forward.

Locate The User Extensions Folder

In the Files & Memory preferences pane you can find the full path to the User Extensions folder. Open this folder in your system file browser.

Add Plugin to Plugins Folder

In the User Extensions folder (1) you will find (or you need to create) a plugins folder (2). Place the glxApplicationProperties.rev plugin stack (3) in this folder and then launch Revolution.

Access Plugin From the Development Menu

Once you launch Revolution you can access the plugin from the Development menu.
